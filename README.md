# Calculadora Básica con Tkinter 

Calculadora Básica con Tkinter 

 Objetivo del Proyecto

Proyecto desarrollado por un equipo de 6 integrantes (4 Devs + 2 Roles de Gestión) para crear una Calculadora Aritmética Básica con Interfaz Gráfica de Usuario (GUI) en Python, siguiendo la metodología Scrum.

La arquitectura modular garantiza que cada desarrollador haya contribuido a una capa específica del sistema, cumpliendo con el Criterio de Terminado (DoD) de la división de trabajo.
